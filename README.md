# README

## Start services

```bash
docker-compose up
```

## Backend

To start the container:

```bash
docker run --rm -p 3000:3000 marches-publics-backend
```

## Frontend

To start the container:

```bash
docker run --rm -p 3000:3000 marches-publics-frontend
```
